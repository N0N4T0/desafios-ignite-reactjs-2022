## Instructions

Após download do projeto, execute o comando abaixo para instalar as dependências.

```
npm i
```

Feito isso execute o comando abaixo para iniciar o projeto.

```
npm run dev
```

O pojeto irá iniciar em:

```
http://localhost:3000/
```
